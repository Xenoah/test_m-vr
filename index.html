開発者向け / SETUP

聖域への入り方

このリポジトリは Unity プロジェクトを含みます。Unity Hub では ルートではなく Unity プロジェクト本体のフォルダーを開いてください。

必要環境

  • Git
  • Unity Hub
  • Unity 6000.5.1f1
  • Windows Build Support / IL2CPP Support

初回セットアップ

  1. リポジトリを clone する
  2. Unity Hub でプロジェクト本体フォルダーを開く
  3. 初回インポートの完了を待つ
  4. 起動シーン initialization.unity を開く
  5. Play モードで起動確認する
git clone https://github.com/Xenoah/XenuyuVR.git
cd Basis_XenuyuVR
# Unity Hub でプロジェクト本体フォルダーを開く

詳細な手順（Windows ビルド・起動オプション等）は 本体リポジトリの README を参照してください。

利用規約 / RULES

マアトの秤 — 守るべき理

成人向けコンテンツは、関係するすべての参加者が成人であり、内容を理解したうえで明確に同意している場合のみ許可されます。

同意のない成人向け表現、未成年が関係する成人向け表現、強要・嫌がらせ・なりすまし・無断撮影や無断共有を含む行為は禁止します。

各ユーザーは、自分が参加する地域の法律、プラットフォーム規約、サーバーやワールドごとのルールを守る必要があります。
