# MedjedVR 公式サイト / Wiki

XenuyuVR（ゼヌユVR）の公式案内・Wiki的サイト。
本体リポジトリ: https://github.com/Xenoah/MedjedVR

## コンセプト

古代エジプト『死者の書』に登場する**謎の神メジェド（Medjed）**をモチーフにした、
**Misskey.io 風**の親しみやすいデザイン。

- 表示名は「**メジェドVR**」（カタカナ）
- キャラクター: メジェドを丸文字・アニメ調のマスコット化（ぱっちり目・ほっぺ・お口）
  - **歩く → つまずく → 転ぶ → 起き上がる**ループアニメ＋クラクラ星
- デザイン言語（Misskey.io の本物のテーマトークン準拠 = **ダークテーマ**）:
  - 背景 `rgb(12,18,16)` / パネル `rgb(25,35,32)` / パネル強調 `rgb(31,44,40)`
  - アクセント（ビビッドライム）`rgb(180,233,0)`、ボタンはライム→緑グラデ `rgb(180,233,0)→rgb(102,233,0)`、文字 `rgb(222,231,228)`
  - リンクはティール `rgb(36,215,206)`、ハッシュタグ橙 `rgb(255,145,86)`
  - 角丸パネル（radius 12px）＋細いディバイダー `rgba(231,255,251,.14)`、ピル型ボタン
  - フォント: **LINE Seed JP**（見出し・ロゴ）／ **Noto Sans JP**（本文）
- 隠し味として『死者の書』らしいヒエログリフの薄いウォーターマーク＋星の背景

## 対象プロジェクト概要

- **名称**: MedjedVR（メジェドVR）
- **種別**: チル・ゲーム対応 VRSNS（Basis フレームワーク製）
- **プラットフォーム**: Windows / VR（OpenVR・OpenXR）
- **技術**: Unity 6000.5.1f1 / C#
- **方向性**: ゆったり過ごせるソーシャルVR ＋ ネットワーク対応ゲーム体験

## 技術スタック（本サイト）

- 静的サイト（HTML / CSS / Vanilla JS）— ビルド不要
- GitHub Pages でのホスティングを想定
- 外部依存なし（フォントは Web フォント、それ以外は自前 SVG）
